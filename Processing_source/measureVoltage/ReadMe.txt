Работа скетча "measureVoltage.pde".
1. Загрузить прошивку на МК
2. Запустить скетч
3. Выбрать тип измерения. CHARGE/DISCHARGE
4. В каталоге со скетчем будут созданы файлы "measureCharge.csv" и "measureDisCharge.csv" с результатами измерений. При каждом запуске соответствующий файл перезаписывается.

Условия
Перед отправкой команды 'v'(включение контроля напряжений) необходимо проинициализировать коридоры значений.
Измерение напряжений не доступно после отправки команды 'v'.
Если коридоры не были проиниициализированы, то после включения проверки командой 'v' и перезагрузке МК произойдет форматирование т.к. все значения будут равны 255(0xFF).

Порядок действий при настройке проверки "контроль напряжений"
1. Загрузить прошивку на МК
2. Отправить на МК значения коридоров последоватьельность из 32 байт.
Пример последовательности:
{  	0,    40,   50,   60,   70,   80,   90,   100,   // chargeMin
	70,   140,  150,  160,  170,  180,  180,  180,   // chargeMax
	200,  200,  200,  200,  200,  200,  190,  190,   // dischargeMin
	255,  255,  255,  255,  255,  255,  255,  255};  // dischargeMax
3. Отправить на МК команду 'v'(активация проверки). Проверка включается 1 раз(флаг в EEPROM) и действует всегда.
